V0.2 of DMTesseracT Ltd's Rpg dialog system developed for the War of Being video game, currently being developed in unreal engine 5.6.

This is made available for free, as is, for evaluation or use in your unreal projects.  No attribution is required, but feedback is always appreciated! 

Free to use in personal and commercial Unreal Engine projects.
Redistribution, resale, or claiming authorship of this plugin (or modified versions) is prohibited.

Please note that this source code is actively a WIP and we would not advise shipping a product without vigorous testing. No guarantees are made that this repository will be updated with changes and fixes throughout internal development, and is made accessible mostly from an educational standpoint. 
Having spent countless hours gathering the information required to make custom EdGraphs, we felt it warranted to add resources to the Internet in the hope it helps shed some light on the process of creating custom assets and editors.
